<!-- Getting Started -->
## 	:toolbox: Getting Started

<!-- Run Locally -->
### :running: Run Locally

Clone the project

```bash
  git clone https://github.com/mhrehman17/decentai.git
```

Go to the download location, and run following command.

```bash
  python -m decentai.main 
```

Now you can see an MNIST example running on screen.
